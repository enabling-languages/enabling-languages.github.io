Welcome to Enabling Languages Opensource Initiative.

Enabling Languages is a loose affiliation of developers working on minority language support on computers, mobile devices and the web. Support and guidance is provided by Enabling Languages staff.

Current projects

  1. Cham Unicode: Tools and resources for Western and Eastern Cham.
  2. Tai languages: support for tai Aiton and Tai Phake, languages of Assam, India that are written in the Myanmar script.
  3. MLWT: a project to provide basic information and resources on Multilingual Web Typography
  4. MyanmarWeb: a resource for imporving myanmar web development, typesetting and typography in langauges that use the Myanmar script.
